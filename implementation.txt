 Refactor dua komponen `sidebar.vue` dan `header.vue` agar layout dashboard konsisten: sidebar tetap fixed di sisi kiri layar, sementara header dan konten utama sejajar di kanan sidebar.

 **Tujuan:**

 * Sidebar tidak berubah posisinya (tetap `fixed`, tinggi layar penuh).
 * Header muncul di sebelah kanan sidebar, bukan di atasnya.
 * Saat sidebar dikompres (collapsed), header menyesuaikan lebar sidebar dengan transisi halus.
 * Breadcrumb dan toggle sidebar muncul di area header bagian kiri, tidak tertutupi sidebar.

 **Instruksi Refactor:**

 1. Pada `sidebar.vue`:

    * Tetap gunakan `position: fixed` dan `h-screen`.
    * Tambahkan z-index `z-20` untuk memastikan sidebar berada di bawah header dropdown, tetapi tidak menutupi header utama.
    * Pastikan class `w-64` (expanded) dan `w-20` (collapsed) tetap digunakan, dikontrol dengan `isCollapsed`.
    * Tambahkan event emit `@toggle-collapse` agar state collapse dapat disinkronkan dengan layout utama.
 2. Pada `header.vue`:

    * Tambahkan prop `isCollapsed` untuk menerima status dari sidebar.
    * Tambahkan binding dinamis pada header:

      ```html
      <header 
        class="bg-white border-b border-gray-200 shadow-sm sticky top-0 z-30 transition-all duration-300"
        :class="[isCollapsed ? 'md:ml-20' : 'md:ml-64']"
      
      ```
    * Pastikan breadcrumb dan tombol toggle sidebar tetap berada di sisi kiri header.
    * Pastikan header memiliki lebar penuh area konten (bukan 100vw).
 3. Di layout utama (misalnya `AdminLayout.vue`):

    * Bungkus dengan flex horizontal:

      ```html
      <div class="flex min-h-screen"
        <Sidebar :sidebar-open="sidebarOpen" @toggle-collapse="isCollapsed = !isCollapsed" /
        <div class="flex-1 flex flex-col w-full transition-all duration-300"
          <Header :is-collapsed="isCollapsed" @toggle-sidebar="sidebarOpen = !sidebarOpen" /
          <main class="flex-1 bg-gray-50 p-6 transition-all duration-300"
            :class="[isCollapsed ? 'md:ml-20' : 'md:ml-64']"
            <slot /
          </main
        </div
      </div
      ```
 4. Gunakan transisi `transition-all duration-300` pada margin atau lebar agar pergeseran terasa halus saat sidebar dibuka/tutup.

 **Hasil yang diharapkan:**

 * Sidebar tetap berada di sisi kiri layar sepanjang waktu.
 * Header dan konten utama berada di sebelah kanan sidebar, sejajar secara horizontal.
 * Saat sidebar dikompres, area header dan konten melebar otomatis tanpa â€œmelompat.â€
 * Breadcrumb dan toggle sidebar selalu tampak di dalam header di posisi kiri atas area konten.
